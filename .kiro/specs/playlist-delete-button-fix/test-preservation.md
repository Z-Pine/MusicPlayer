### 保留属性定义
**Property 2: Preservation** - 对于所有不涉及点击未选中歌曲删除按钮的操作，修复后的代码应该产生与原始代码完全相同的行为。

形式化表示：
```
FOR ALL input WHERE NOT isBugCondition(input) DO
  ASSERT removeSongs_original(input) = removeSongs_fixed(input)
END FOR
```

其中 `isBugCondition(input)` 定义为：
```
input.clickSource = 'delete-button'
AND input.isSelected = false
AND selectedSongIds.size = 0
```
