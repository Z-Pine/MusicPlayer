## 修复总结

### Bug 描述
在歌单的歌曲列表界面，歌曲最右边的删除按钮（❌）点击无响应，没有执行删除操作。

### 根本原因
1. 删除按钮未传递歌曲 ID 参数
2. `removeSongs` 函数依赖 `selectedSongIds` 集合，当集合为空时直接返回
3. 函数缺少处理"删除指定歌曲"和"删除选中歌曲"两种场景的逻辑

### 修复方案
修改 `f:\VSCode Projects\MusicPlayer\src\views\PlaylistView.vue`：

1. **函数签名修改**：
   ```typescript
   async function removeSongs(songId?: number)
   ```

2. **删除逻辑修改**：
   ```typescript
   // 优先删除选中的歌曲（批量删除），否则删除指定的歌曲
   let idsToRemove: number[];
   if (selectedSongIds.value.size > 0) {
     // 如果有选中的歌曲，删除所有选中的（批量删除）
     idsToRemove = Array.from(selectedSongIds.value);
   } else if (songId !== undefined) {
     // 如果没有选中的歌曲，但提供了 songId，删除该歌曲
     idsToRemove = [songId];
   } else {
     // 既没有选中的歌曲，也没有提供 songId，直接返回
     return;
   }
   ```

3. **删除按钮调用修改**：
   ```vue
   <button @click.stop="removeSongs(song.id)">✕</button>
   ```
